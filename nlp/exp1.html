Natural Language Processing - Experiment 2
Word-Level Text Analysis using NLTK

Aim

To perform word-level analysis on natural language text using tokenization, normalization, stop-word removal, frequency analysis, stemming, lemmatization, and part-of-speech (POS) tagging.

Software Used

  • Python 3.10+
  • NLTK Library
  • Collections Module
  • Jupyter Notebook / Google Colab / VS Code

Theory

Word-level preprocessing is an essential step in Natural Language Processing (NLP). It transforms raw text into a structured form by performing tokenization, normalization, stop-word removal, frequency analysis, stemming, lemmatization, and POS tagging. These operations help reduce noise, identify important words, and prepare textual data for tasks such as text classification, sentiment analysis, and information retrieval.

Input Code

import nltk
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords
from nltk.stem import PorterStemmer, WordNetLemmatizer
from nltk import pos_tag
from collections import Counter

# Download required resources (first time only)
nltk.download('punkt')
nltk.download('stopwords')
nltk.download('wordnet')
nltk.download('averaged_perceptron_tagger')

text = """Natural Language Processing allows computers to understand
human language. NLP techniques are widely used in text analysis
and machine learning."""

# Tokenization
tokens = word_tokenize(text)

# Normalization & Stop-word Removal
stop_words = set(stopwords.words('english'))
words = [w.lower() for w in tokens if w.isalpha()]
filtered = [w for w in words if w not in stop_words]

# Frequency Analysis
freq = Counter(filtered)

# Stemming
stemmer = PorterStemmer()
stems = [stemmer.stem(w) for w in filtered]

# Lemmatization
lemmatizer = WordNetLemmatizer()
lemmas = [lemmatizer.lemmatize(w) for w in filtered]

# POS Tagging
tags = pos_tag(tokens)

print("Filtered Words:", filtered)
print("Frequency:", freq)
print("Stemmed:", stems)
print("Lemmatized:", lemmas)
print("POS Tags:", tags)

Observation

Task Observation
Tokenization Paragraph split into individual words.
Stop-word Removal Common words like "the", "to", and "and" removed.
Frequency Analysis Repeated content words identified.
Stemming & Lemmatization Stemmer shortened words, lemmatizer returned valid root forms.
POS Tagging Words assigned grammatical tags (NN, VB, JJ, etc.).

Result / Conclusion

The word-analysis pipeline was successfully implemented using NLTK. The experiment demonstrated tokenization, normalization, stop-word removal, frequency analysis, stemming, lemmatization, and POS tagging, providing meaningful lexical information for further NLP applications.
