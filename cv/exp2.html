Computer Vision - Experiment 2
Basic Image Processing using Python and OpenCV

Aim

To implement basic image processing functions using Python and OpenCV.

Software Used

  • Python 3.x
  • OpenCV (cv2)
  • NumPy
  • VS Code / PyCharm / Spyder

Theory

Image processing involves manipulating digital images to improve their quality or extract useful information. OpenCV provides built-in functions for common image processing tasks such as image reading, displaying, grayscale conversion, resizing, rotation, and saving processed images. These operations form the foundation for advanced computer vision applications like object detection, image recognition, and image segmentation.

Input Code

import cv2

# Read image
img = cv2.imread("input.jpg")

# Display original image
cv2.imshow("Original Image", img)

# Convert image to grayscale
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
cv2.imshow("Grayscale Image", gray)

# Resize image
resized = cv2.resize(img, (400, 300))
cv2.imshow("Resized Image", resized)

# Rotate image by 90 degrees clockwise
rotated = cv2.rotate(img, cv2.ROTATE_90_CLOCKWISE)
cv2.imshow("Rotated Image", rotated)

# Save grayscale image
cv2.imwrite("gray_image.jpg", gray)

cv2.waitKey(0)
cv2.destroyAllWindows()

Observation

Operation Observation
Image Reading Input image loaded successfully.
Grayscale Conversion Color image converted to grayscale.
Image Resizing Image resized to 400 × 300 pixels.
Image Rotation Image rotated 90° clockwise.
Image Saving Processed grayscale image saved as gray_image.jpg.

Result / Conclusion

Basic image processing operations including image reading, displaying, grayscale conversion, resizing, rotation, and image saving were successfully implemented using Python and OpenCV. The processed images were generated correctly, demonstrating the fundamental capabilities of OpenCV for image manipulation.
