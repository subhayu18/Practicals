Computer Vision - Experiment 1
Find the Frame Rate (FPS) of a Video using OpenCV

Aim

To develop a Python program using OpenCV to determine the frame rate (FPS) of a video and display the FPS value on the video during playback.

Software Used

  • Python 3.x
  • OpenCV Library (cv2)
  • VS Code / PyCharm / Spyder
  • Input Video File (.mp4)

Theory

Frame Rate (FPS - Frames Per Second) indicates the number of frames displayed every second in a video. A higher FPS results in smoother video playback. OpenCV provides the CAP_PROP_FPS property to retrieve the FPS of a video file. The obtained value can be displayed on each frame using the cv2.putText() function while the video is being played.

Input Code

import cv2

# Open the video file
video = cv2.VideoCapture("input_video.mp4")

# Get frame rate
fps = video.get(cv2.CAP_PROP_FPS)

print("Frame Rate (FPS):", fps)

while True:
    ret, frame = video.read()

    if not ret:
        break

    # Display FPS on the frame
    cv2.putText(frame,
                f"FPS: {fps:.2f}",
                (20, 40),
                cv2.FONT_HERSHEY_SIMPLEX,
                1,
                (0, 255, 0),
                2)

    cv2.imshow("Video with FPS", frame)

    if cv2.waitKey(25) & 0xFF == ord('q'):
        break

video.release()
cv2.destroyAllWindows()

Observation

Parameter Observation
Input Video input_video.mp4
Detected FPS 30.00 FPS (example)
Console Output Frame Rate (FPS): 30.0
Video Output FPS value displayed on every frame

Result / Conclusion

The frame rate (FPS) of the video was successfully extracted using OpenCV and displayed on every frame during playback. The experiment demonstrates how to read video properties and overlay text information on video frames using Python.

  • Retrieve video properties using CAP_PROP_FPS.
